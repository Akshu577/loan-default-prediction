# Loan Default Prediction

A machine learning project to predict loan defaults using classification modeling.

## Features
- Binary classification with threshold optimization
- SHAP explainability for feature importance
- Key features: late_payments, credit_score, risk_score, debt_ratio

## Results
- AUC: 0.91
- Best Threshold: 0.1 (F1: 0.876)
- Accuracy: 83%

## Setup
```bash
pip install -r requirements.txt
python main.py
```

# Loan Default Prediction

## Overview
This machine learning project predicts whether a borrower will default on a loan using financial and behavioral features.

## Features Used
- Income
- Credit Score
- Debt Ratio
- Late Payments
- Loan Amount

## Technologies
- Python
- Pandas
- Scikit-learn
- XGBoost
- SHAP
- Matplotlib

## Workflow
1. Generate dataset
2. Preprocess data
3. Train XGBoost model
4. Evaluate performance
5. Explain predictions using SHAP

## Results
- Accuracy: 80%
- F1 Score: 0.83
